# Countries Wiki

- Project for view countries details (flags, names, borders, area...) with [REST Countries API](https://restcountries.com)
- In project used:
  - ShadCN UI
  - motion (animation library)
  - Tailwind
  - FSD
  - Redux (RTK + RKT Query)
  - React
  - React Router
  - React Hook Form
  - Vite
  - SCSS
